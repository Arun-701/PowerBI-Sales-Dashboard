## 👨‍💻 Created by

## 📌 Project Overview

This project is a Sales & Profit Analytics Dashboard built using Power BI.
It helps analyze business performance using interactive visuals.

## 🚀 Key Features

* Total Sales, Profit, Orders KPIs
* Monthly Sales Trend
* Platform-wise analysis (Amazon, Flipkart, Meesho)
* Category-wise distribution
* State-wise sales analysis

## 🛠 Tools Used

* Power BI
* Excel / CSV Dataset
